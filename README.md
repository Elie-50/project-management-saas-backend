# Team Management App

A team management application built with **NestJS** and **TypeScript**. This app allows users to manage organizations, teams, projects, and tasks with advanced features for collaboration, task assignment, and data management.

### Features:

* **Organization Management**: Create and manage organizations.
* **Team Collaboration**: Assign members to organizations, manage roles and permissions.
* **Project Management**: Create and track projects, assign tasks to members.
* **Task Management**: Manage tasks with assignees, deadlines, and statuses.
* **User Authentication**: Secure authentication and authorization for users.

---

## Project Setup

To set up the project locally:

```bash
$ npm install
```

---

## Compile and Run the Project

### Development:

```bash
$ npm run start
```

### Watch Mode (Auto-reload on changes):

```bash
$ npm run start:dev
```

### Production Mode:

```bash
$ npm run start:prod
```

---

## Run Tests

### Unit Tests:

Run unit tests to ensure individual components and services work as expected.

```bash
$ npm run test
```

### e2e Tests:

Run end-to-end tests to verify that the entire flow of the app works with real database interactions.

```bash
$ npm run test:e2e
```
